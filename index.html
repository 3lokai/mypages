üìñ Complete Usage Guide

  1. Go to attendee list page - Log in if needed, navigate to the page showing all attendees/profiles
  2. Click the bookmarklet - The scraper UI appears in the top-right corner
  3. Click "Select Profile Link" - Then click on ANY attendee's name (even if it's a hyperlink)
  4. Wait for collection - Scraper automatically clicks "Load More" and collects all profile URLs
  5. Click "Configure Fields" - Opens the first profile as a sample
  6. Add fields - Click "+ Add Field", enter field name (e.g., "Name"), click "Select", then click the element on the profile
  7. Repeat for all fields - Add Title, Company, Location, Budget, Interests, etc.
  8. Click "Start Deep Scrape" - Confirms the number of profiles and estimated time
  9. Let it run - Keep the tab open. Watch the progress bar. It will visit each profile automatically
  10. Download CSV - When complete, CSV downloads automatically with all scraped data

üîß Troubleshooting

Q: Bookmarklet not working?
A: Try right-clicking the button and selecting "Bookmark this link". Or create a bookmark manually and paste the JavaScript URL.
Q: It's not finding any profiles?
A: Make sure you clicked on an actual link (<a> tag) that goes to profile pages. Try clicking directly on the attendee's name.
Q: Fields are coming up empty?
A: The selectors might not be robust. Try selecting the direct text element, not a parent container. Test with 2-3 profiles first.
Q: Can I pause and resume?
A: Currently no. The scraper needs to run continuously. For large scrapes (500+), consider running overnight.
